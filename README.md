## Project setup

### Install dependences
```
$ npm install
```

### Compiles fake api
```
$ cd server
$ json-server --watch db.json --routes routes.json
```

### Finally
```
$ npm run serve
```

See [Frontend deployed](https://pry-asimov-cybersoft-21.web.app/).
